Badges & Notifications Guide: This page contains beautiful badges, buttons, and notification icons. Everything here runs entirely on Pure CSS! There is absolutely No JavaScript required. Follow these 3 simple steps to copy them.
Step 1: Global Variables (Mandatory)

To ensure the colors and shadows match perfectly, your project must have the Root Variables (:root { --color-primary... }) defined in your global style.css.

Step 2: Copy the HTML

Open pages/components/userinterface/badges.html. Ignore the .demo-box and .demo-preview-area wrappers (they are just for the showcase). Only copy the core HTML snippet inside them. 

<!-- Example: Animated Ringing Bell -->
<div class="notif-wrapper">
  <i class="bi bi-bell-fill icon-base-5"></i>
  <span class="badge-notif-5"></span>
</div>
Step 3: Copy CSS (Base + Specific)

Open assets/css/component/userinterface/badges.css. You need to copy two things to prevent the layout from breaking:

  • A. Base Utilities: Copy the base wrapper class from the top of the CSS file (e.g., .badge-base for simple badges, .btn-wrapper & .btn-base for buttons, or .notif-wrapper for icons).
  • B. Specific Style: Scroll down to the specific badge/button you want and copy its dedicated CSS block (e.g., .badge-glass or .icon-base-5). 
/* Example: Copying a Notification Icon */

/* 1. Base Utility */
.notif-wrapper { position: relative; display: inline-flex; ... }

/* 2. Specific Style */
.icon-base-5 { font-size: 1.5rem; ... }
@keyframes swing { ... }
No JavaScript Needed!

Even the complex animations (like the ringing bell or glowing neon effects) are built using CSS @keyframes. Simply copy the HTML and CSS, and it works instantly.
