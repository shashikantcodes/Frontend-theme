Accordion Guide: This page contains 15 unique accordion styles. They run entirely on Bootstrap's native collapse plugin + Custom CSS. No custom JS file is required! Follow these 4 steps to copy them perfectly.
Step 1: Global Setup (Bootstrap JS is Mandatory)

Accordions rely on JavaScript to open and close. Ensure your project has the core files loaded:

  • Scripts: You MUST have Bootstrap's JS bundle (bootstrap.bundle.min.js) loaded via your components/scripts.html.
  • Head Resources: Bootstrap 5 CSS, Bootstrap Icons, and Root Variables (style.css) must be loaded.
Step 2: Copy the "Common Utils" CSS (Once per project)

Open assets/css/component/userinterface/accordians.css. At the very top, there is a section called COMMON UTILS & ANIMATIONS. This contains the magic for the animated plus/minus icons (.smart-icon). Copy this block ONCE. 

/* --- SMART ICON SYSTEM --- */
.smart-icon { position: relative; ... }
.accordion-button:not(.collapsed) .smart-icon { transform: rotate(180deg); }
Step 3: Copy the HTML

Find the accordion style you want and copy its HTML wrapper. Pay attention to the IDs (like #acc1 and #c1_1)—they must match for the accordion to work. 

<!-- Example from accordian.html -->
<div class="acc-theme-saas style-process">
  <div class="accordion" id="acc1">
    <!-- Accordion Items Here -->
  </div>
</div>
Step 4: Copy Specific Variation CSS

Go back to accordians.css and scroll down to your specific style block. Copy ONLY that block to keep your CSS clean. 

/* =========================================
   STYLE 1: WORKING PROCESS 
   ========================================= */
.acc-theme-saas.style-process .accordion-item { ... }
.acc-theme-saas.style-process .accordion-button { ... }
